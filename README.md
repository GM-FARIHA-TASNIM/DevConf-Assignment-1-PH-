# DevConf 2026

A modern and responsive developer conference landing page built using **HTML5** and **CSS3**. This project was created as part of the Programming Hero Web Development assignment. The website promotes **DevConf 2026**, featuring conference speakers, pricing plans, a special Hackathon Challenge section, and a professional footer.

---

## Features

- Responsive Navigation Bar
- Hero Banner with Background Image and Overlay
- Meet the Speakers Section
- Pricing Section with Featured Pro Plan
- AI-Generated Hackathon Challenge Section
- Professional Footer with Social Media Icons
- Clean and Semantic HTML Structure
- Built using only HTML and CSS

---

## Technologies Used

- HTML5
- CSS3
- Font Awesome
